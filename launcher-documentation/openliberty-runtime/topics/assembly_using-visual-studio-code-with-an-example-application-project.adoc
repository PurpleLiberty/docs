= Using Visual Studio Code with an example application project

link:https://code.visualstudio.com[Visual Studio Code] is an integrated development environment, or IDE, that provides tooling for writing software.
You can use it to make changes to the code of your example application.

.Prerequisites

* Have an example application deployed to {OpenShiftOnline}, or {OpenShiftLocal} using the _Deploy in OpenShift Online_ option.
* Have link:https://code.visualstudio.com[Visual Studio Code] downloaded and installed.
